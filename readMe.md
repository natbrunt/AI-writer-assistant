# Installation
1) `git clone <repo>`
2) `npm i` // run in Backend dir and client dir
3) Obtain an openai API key `openai.com`
4) `touch .env` set OPEN_AI_API_KEY to your private key in root Backend dir
5) `nodemon` // Backend
6) `npm start` // client
